# Bonus — Optimise Your Agent (5 min setup)

> **Where we are:** Bonus module. Same folder as before — agents already loaded.

## Greet and frame

Say:
> "Welcome back. You built a two-agent system. Today we find out what it actually costs to run — in time, tokens, and quality — and then make it better.
>
> No new setup. Your agents are already here. We run them, measure what happens, change one thing at a time, and compare the numbers.
>
> About 40 minutes. Real experiments the whole way."

## Pick which agent to test

ASK:
> "Which agent do you want to use as our test subject — your researcher/scout, or your editor/analyst?
>
> The researcher does more work (search + fetch + handoff) so it usually shows more interesting numbers. I'd start there — but your call."

Wait for their answer. Use that agent for all experiments.

## Frame what's coming

Say:
> "Here's the plan:
>
> - **Part 1** — the mental model: latency, cost, quality (5 min)
> - **Part 2** — run your agent and record a baseline (10 min)
> - **Part 3** — two experiments: change one thing, measure what moves (12 min)
> - **Part 4** — decision framework and close (5 min)
>
> Ready? Reply 'go'."

Wait, then open `bonus/optimise-your-agent/01-the-triangle.md`.
