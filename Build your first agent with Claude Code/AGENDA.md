# The 40-Minute Plan

You're going to walk out of this hour confident in two things:

1. **Building agents** — you'll have built a pair of agents that work together, and you'll understand the brain (LLM + system prompt), the hands (tools), and the loop.
2. **Using Claude Code** — many of you are seeing this for the first time. By the end you'll have used it to read files, edit them, run agents, and watch them work.

This activity is **self-paced**. June (your tutor) walks you through it.

---

## Welcome (5 min)
Pick your mission, your topic, and the templates get set up.

## Lesson 1 — The brain (10 min)
Look at the system prompt that runs your agent. Understand what it tells the LLM to do. Tweak it if you want.

## Lesson 2 — The hands (6 min)
Look at the tools your agent has. Understand why each one. Adjust if it makes sense.

## Lesson 3 — The handoff (8 min)
Meet the second agent. See how the first calls the second when it needs help. **This is agent-to-agent.**

## Lesson 4 — Run it (8 min)
Invoke your agent. Watch the loop. Watch the handoff. See the output file appear.

## Lesson 5 — Reflect (3 min)
Name what you built and where to go next.

---

## What you'll have at the end

- Two subagents living in `.claude/agents/` you can invoke any time, on any topic
- An output file in `output/` from a topic *you* picked
- A clear mental model: brain (LLM + prompt) + hands (tools) + loop + handoff = agent
