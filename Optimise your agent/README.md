# Optimise Your Agent

A 40-minute bonus module for people who've already built a multi-agent system.

You built two agents that work together. Now you're going to find out what they're actually costing — in time, money, and quality — and learn how to improve all three.

---

## Before you start

You need the agents from the "Build Your First Agent" module. They should be in `.claude/agents/` inside that folder. If you haven't done that module yet, start there first.

---

## Setup

Open this folder in Claude Code:

```
claude
```

Then say:

> **let's start**

June will take it from there.

---

## What you'll do

- Run your existing agents and see exactly what they're spending (tokens, latency)
- Learn the triangle: latency, cost, quality — and why you can't max all three
- Run two live experiments on your own agents
- Walk out with a decision framework: which lever to pull, when, and why

---

## What you'll have at the end

- Real performance numbers for your own agents
- An edited, faster or cheaper version of at least one agent
- A mental model for every AI performance decision you'll ever make
