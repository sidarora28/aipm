# Optimise Your Agent — Setup Guide

You've already built your agents. Now you're going to find out what they're actually costing and how to make them faster, cheaper, and better.

This guide gets you set up in under 5 minutes.

---

## Before you start

Make sure you have:
- ✅ Completed the "Build Your First Agent" module
- ✅ Your two agents still in `.claude/agents/` inside that folder
- ✅ A Claude Pro subscription (same one you used before)

That's it. No new tools, no new accounts.

---

## Step 1 — Get the new folder

**If you downloaded the ZIP last time:**

1. Go to **https://github.com/sidarora28/aipm**
2. Click the green **Code** button → **Download ZIP**
3. Unzip it — you'll see a folder called `aipm-main`
4. Inside it, find the folder called **`Optimise your agent`**

**If you cloned the repo with git:**

Open your terminal and run:
```
cd ~/aipm
git pull
```

You'll see `Optimise your agent` appear in the folder.

---

## Step 2 — Open the folder in Claude Code

**Claude Desktop:**

Open Claude Desktop → File → Open Folder → select the `Optimise your agent` folder.

**Claude Code CLI:**

```
cd "~/aipm/Optimise your agent"
claude
```

(Adjust the path to wherever you unzipped the folder.)

---

## Step 3 — Start

Type this in the chat:

```
let's start
```

June takes over from there. She'll check your agents are ready, then walk you through everything.

---

## Something broken?

| Problem | Fix |
|---|---|
| Can't find my agents | Open the "Build Your First Agent" folder in Claude Code first and check `.claude/agents/` — they should be there. June can also help you copy them across. |
| June seems confused | Type `I'm stuck` — she'll help. |
| Want to start over | Type `/clear`, then `let's start`. |

---

## What you'll do in this module

- Run your existing agents and record real performance numbers
- Pull two levers — context trim and prompt tightening — and see what moves
- Walk out with a decision framework for every AI performance call you'll ever make

**~40 minutes. Same agents you already built.**
